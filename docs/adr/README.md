# Architecture Decision Records

ADRs dokumentieren dauerhafte technische Entscheidungen. Statuswerte sind `Proposed`, `Accepted`, `Superseded` und `Rejected`. Änderungen ersetzen frühere Entscheidungen nicht stillschweigend, sondern referenzieren sie.

Jede ADR beschreibt Status, Kontext, Entscheidung, Begründung, Alternativen, positive und negative Folgen, Sicherheitsauswirkungen und spätere Überprüfung.

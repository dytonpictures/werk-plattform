# ADR-0001: Ubuntu LTS als Referenzplattform

- **Status:** Accepted
- **Kontext:** Self-Hosting benötigt eine stabile, verbreitete Basis.
- **Entscheidung:** Ubuntu LTS ist die Referenzplattform.
- **Begründung:** Lange Pflege, breite Dokumentation und gute Docker-Unterstützung.
- **Alternativen:** Debian, andere Linux-Distributionen.
- **Positive Folgen:** Reproduzierbare Betriebsanleitungen.
- **Negative Folgen:** Andere Systeme sind nicht primär getestet.
- **Sicherheitsauswirkungen:** Sicherheitsupdates der LTS-Version müssen zeitnah installiert werden.
- **Überprüfung:** Bei Auslaufen der verwendeten LTS-Version.

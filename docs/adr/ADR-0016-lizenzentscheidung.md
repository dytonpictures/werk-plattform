# ADR-0016: Lizenzentscheidung für Open-Source-Veröffentlichung

- **Status:** Proposed
- **Kontext:** WERK soll offen entwickelt werden, eine Lizenz ist noch nicht verbindlich gewählt.
- **Entscheidung:** Apache-2.0 und AGPL-3.0 werden vor Veröffentlichung rechtlich und strategisch bewertet.
- **Begründung:** Apache-2.0 fördert breite Nutzung; AGPL-3.0 schützt Netzwerkbereitstellungen stärker als Copyleft-Beitrag.
- **Alternativen:** Andere OSI-Lizenzen oder duale Lizenzierung.
- **Positive Folgen:** Die Entscheidung bleibt transparent statt stillschweigend angenommen.
- **Negative Folgen:** Das Repository ist bis zur Lizenzdatei nicht rechtlich als Open Source freigegeben.
- **Sicherheitsauswirkungen:** Keine unmittelbaren; Abhängigkeitslizenzen müssen trotzdem geprüft werden.
- **Überprüfung:** Vor erster öffentlicher Veröffentlichung.

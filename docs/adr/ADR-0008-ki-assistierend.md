# ADR-0008: KI zunächst assistierend und menschlich kontrolliert

- **Status:** Accepted
- **Kontext:** KI kann helfen, darf aber keine unkontrollierten Geschäftsänderungen ausführen.
- **Entscheidung:** KI liefert zunächst Vorschläge; Aktionen erfordern bewusste Freigabe.
- **Begründung:** Nachvollziehbarkeit und Risikobegrenzung.
- **Alternativen:** Keine KI, autonome Agenten.
- **Positive Folgen:** Kontrollierte Einführung.
- **Negative Folgen:** Weniger Automatisierung.
- **Sicherheitsauswirkungen:** Anbieter, Berechtigung, Datenfluss und Audit müssen explizit sein.
- **Überprüfung:** Vor jeder KI-Integration.

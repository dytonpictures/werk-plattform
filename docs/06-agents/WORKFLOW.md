# Agenten-Workflow

## Rollen

Orchestrator, Architect, Backend, Frontend, Design System, Database, Security, DevOps, QA, Reviewer und Documentation arbeiten innerhalb ihrer fachlichen Verantwortung.

## Ablauf

Tasks wechseln durch `BACKLOG`, `READY`, `IN_PROGRESS`, `BLOCKED`, `REVIEW` und `DONE`. Implementierende Agenten geben ihre Arbeit mit dem Handoff-Template weiter. Eine unabhängige Prüfung ist vor `DONE` erforderlich; Selbstfreigabe reicht nicht aus.

Blocker nennen Ursache, Auswirkung, getestete Wege und sichere Alternativen.
